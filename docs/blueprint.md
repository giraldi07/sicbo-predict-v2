# **App Name**: Sicbo Oracle

## Core Features:

- Advanced Pattern Recognition: Uses a proprietary algorithm to identify complex patterns in Sic Bo results for smarter predictions.
- AI-Powered Prediction Tool: Generates size (Big/Small) and parity (Odd/Even) predictions using AI. The tool will use game history to learn patterns and dynamically adjust predictions.
- Smart Bet Management: Suggests betting amounts based on user-defined risk profiles and win/loss history, incorporating the Martingale strategy.
- Customizable Betting Parameters: Allows users to input their initial capital, target profit, and risk tolerance to tailor the AI's betting strategy.
- Real-Time Statistics Dashboard: Displays live win/loss ratios, profit/loss calculations, and pattern hit rates to inform betting decisions.
- Game History Tracking: Maintains a detailed history of dice rolls, predictions, and betting outcomes for user review and algorithm training.
- Foolproof 'Any' (Leopard) Prediction: 100% accurate prediction for the ANY (Leopard) outcome (three identical dice) by tracking when such events become statistically probable based on game history. Note: Since the frequency of these outcomes is mathematically determined, this feature essentially highlights opportune times, not true prediction.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke a sense of sophistication and insight.
- Background color: Dark gray (#212121) to ensure readability and focus on key data.
- Accent color: Teal (#009688) to highlight important stats and betting options.
- Body and headline font: 'Inter', a sans-serif for clear data presentation.  Note: currently only Google Fonts are supported.
- Use simple, data-driven icons to represent win/loss trends and betting options.
- A clean, data-focused layout that emphasizes real-time statistics and AI-driven insights.
- Subtle animations for dice rolls and updated statistics.
